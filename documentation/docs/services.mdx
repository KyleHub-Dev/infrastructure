## Included Services

### Langfuse

LLM observability and prompt management platform.

- **Location**: Homelab
- **Port**: 3000
- **Auth**: OIDC with Zitadel

See [homelab-core/services/langfuse](https://github.com/KyleHub-Dev/infrastructure/tree/main/homelab-core/services/langfuse) for configuration.
